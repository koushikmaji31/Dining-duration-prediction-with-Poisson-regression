# Poisson Regression for Dinner Time Prediction

## Overview

This Python script implements Poisson Regression from scratch to predict dinner completion times based on various features such as day, holiday, meal type, extras, and whether the meal is vegetarian or not. It also includes a comparison with an inbuilt Linear Regression model.
